# Vehicle_Type_Classificaton
A deep learning-based Vehicle Type Classification system built using CNN and TensorFlow to accurately identify and classify vehicle categories from images. The project covers image preprocessing, model training, performance evaluation, confusion matrix analysis, and real-world computer vision applications in smart traffic and automation systems.

**Project Overview**

This project focuses on Vehicle Type Classification using Deep Learning and Convolutional Neural Networks (CNNs) with TensorFlow and Keras. The model is trained on a dataset containing multiple categories of vehicle images and is capable of automatically identifying and classifying different types of vehicles.

The project demonstrates the complete workflow of an image classification system including:

- Dataset extraction and preprocessing
- Data visualization
- Dataset splitting
- CNN model building
- Data augmentation
- Model training and evaluation
- Accuracy visualization
- Confusion matrix generation
- Classification report analysis

**Dataset Information**

The project uses a Vehicle Type Image Dataset containing images of various vehicle categories.

Dataset Features:
-Multiple vehicle categories
-RGB vehicle images
-Organized into separate folders by class
-Suitable for supervised image classification

Dataset Structure: 
vehicle_dataset/
│
├── Vehicle Type Image Dataset (Version 2) VTID2/
│   ├── Bus/
│   ├── Car/
│   ├── Motorcycle/
│   ├── Truck/
│   ├── Bicycle/
│   └── ...
