# Vehicle_Type_Classificaton
A deep learning-based Vehicle Type Classification system built using CNN and TensorFlow to accurately identify and classify vehicle categories from images. The project covers image preprocessing, model training, performance evaluation, confusion matrix analysis, and real-world computer vision applications in smart traffic and automation systems.

Project Overview

This project is a Vehicle Type Classification System built using Deep Learning and Convolutional Neural Networks (CNNs) with TensorFlow and Keras.
The model is trained on a vehicle image dataset to automatically classify different types of vehicles from images. The project demonstrates the complete workflow of a computer vision pipeline including:

Dataset extraction and preprocessing
Data visualization
Dataset loading using TensorFlow
CNN model building
Model training and validation
Performance evaluation
Confusion matrix generation
Classification report analysis

This project is ideal for beginners and intermediate learners who want to understand how image classification works using deep learning.
